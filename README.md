# Security Home Lab

## Overview
This project is a security home lab built with Ubuntu and Windows virtual machines to practice network monitoring, firewall configuration, port scanning, and packet analysis.

The lab simulates a small enterprise-style environment and demonstrates hands-on experience with Linux administration, networking, and cybersecurity fundamentals.

---

## Technologies Used
- Ubuntu Linux
- Windows
- VirtualBox
- Nmap
- Wireshark
- UFW Firewall
- Bash

---

## Lab Environment
- Configured Ubuntu and Windows virtual machines
- Established internal VM networking
- Tested connectivity and IP configuration
- Monitored traffic using Wireshark
- Scanned systems using Nmap
- Applied firewall rules using UFW
- Evaluated system exposure before and after firewall changes

---

## Skills Demonstrated
- Linux system administration
- Network troubleshooting
- Packet analysis
- Port scanning
- Firewall configuration
- Virtualization
- Cybersecurity fundamentals

---

## What I Learned
This project helped me understand how systems communicate across a network, how to identify exposed services, and how monitoring and firewall tools are used to secure systems in real-world IT and security environments.
